# MazeRush

### Enlace del video:

## Objetivo 

Tomar 24 monedas sin dejar que ningun enemigo te alcance.

## Controles
- **W, A, S, D** – Movimiento
- **Space** – Salto
- **Mouse** – Mirar alrededor

## Enemigos y features:
1. Salta
2. Crece 
4. Hace un sonido cuando sigue al jugador
